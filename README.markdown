xcode utility

[![asciicast](https://asciinema.org/a/88109.png)](https://asciinema.org/a/88109)

## Installation

```
pip install xcute
```

## Contribution

```
git clone https://github.com/schwa/xcute.git
cd xcute
pip install -e .
```

## Distribution

```
python setup.py sdist
twine upload --skip-existing dist/*
```
